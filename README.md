## Scheduler by Team Takashimaya

An [online scheduler & shared calendar](https://my-project-1559612665398.firebaseapp.com) built using Vue and FullCalendar, and hosted on Firebase. Users can create projects, add their project members, and easily find out common available dates and timings to schedule meetings. This is a project done for a module, BT3103 Application Systems Development for Business Analytics.
